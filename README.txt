Declares Streamlike as an oEmbed provider.

== Description ==

Streamlike is a carbon neutral SaaS Enterprise Video Platform for live and on-demand streaming: https://www.streamlike.eu

This plugin allows pasting Streamlike media urls into Gutenberg's "embed content" block to have the Streamlike media player displayed seamlessly using oEmbed.

Streamlike terms and conditions: https://www.streamlike.eu/terms-and-conditions 

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'.
2. Search for 'oEmbed Streamlike'.
3. Click on the 'Install Now' button.
4. Activate 'oEmbed Streamlike' plugin.

= From WordPress.org =

1. Download 'oEmbed Streamlike'.
2. Upload the 'oembed-streamlike' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp...).
3. Activate 'oEmbed Streamlike' plugin from your Plugins page.

= Once Activated =

1. No setting: just enjoy!
